## CHANGE 4: Draggable bottom sheet handle on Add Task modal

The drag handle at the top of the Add Task modal 
is currently a static visual element. Make it 
functional using react-native-reanimated and 
react-native-gesture-handler (both already installed).

**Two snap points:**
  HALF = screenHeight * 0.55   // default starting position
  FULL = screenHeight * 0.92   // expanded full screen

**Implementation using Reanimated 2:**

  import { useSharedValue, useAnimatedStyle, 
           withSpring, runOnJS } from 
           'react-native-reanimated';
  import { PanGestureHandler } from 
           'react-native-gesture-handler';
  import Animated from 'react-native-reanimated';

  const screenHeight = Dimensions.get('window').height;
  const HALF = screenHeight * 0.55;
  const FULL = screenHeight * 0.92;

  // translateY: 0 = FULL height, positive = shorter
  // Starting position: sheet shows at HALF height
  const translateY = useSharedValue(screenHeight - HALF);

  const animatedStyle = useAnimatedStyle(() => ({
    transform: [{ translateY: translateY.value }],
  }));

  const gestureHandler = useAnimatedGestureHandler({
    onStart: (_, ctx) => {
      ctx.startY = translateY.value;
    },
    onActive: (event, ctx) => {
      // Follow finger but clamp within valid range
      const newY = ctx.startY + event.translationY;
      translateY.value = Math.max(
        screenHeight - FULL, 
        Math.min(screenHeight - HALF * 0.7, newY)
      );
    },
    onEnd: (event) => {
      // Snap to nearest point based on velocity + position
      const midPoint = screenHeight - 
        ((FULL - HALF) / 2 + HALF);
      
      if (event.velocityY < -500 || 
          translateY.value < midPoint) {
        // Snap to FULL
        translateY.value = withSpring(
          screenHeight - FULL, 
          { damping: 20, stiffness: 200 }
        );
      } else if (event.velocityY > 1000) {
        // Fast downward swipe — dismiss the modal
        translateY.value = withSpring(
          screenHeight, 
          { damping: 20, stiffness: 200 },
          () => runOnJS(onClose)()
        );
      } else {
        // Snap back to HALF
        translateY.value = withSpring(
          screenHeight - HALF, 
          { damping: 20, stiffness: 200 }
        );
      }
    },
  });

**Sheet structure:**

  // The modal backdrop (full screen, tap to close):
  <TouchableWithoutFeedback onPress={onClose}>
    <View style={StyleSheet.absoluteFill} />
  </TouchableWithoutFeedback>

  // The animated sheet panel:
  <Animated.View style={[styles.sheet, animatedStyle]}>
    
    {/* Drag handle area — this is what the user grabs */}
    <PanGestureHandler onGestureEvent={gestureHandler}>
      <Animated.View style={styles.handleArea}>
        <View style={styles.handle} />
        {/* handle = 36×4 rounded pill, color #555 */}
      </Animated.View>
    </PanGestureHandler>

    {/* The form content inside a ScrollView */}
    <ScrollView 
      style={{ flex: 1 }}
      keyboardShouldPersistTaps="handled"
    >
      {/* Title input, urgency pills, date chip, 
          subtasks, Add Task button */}
    </ScrollView>

  </Animated.View>

**Sheet styles:**
  sheet: {
    position: 'absolute',
    left: 0,
    right: 0,
    bottom: 0,
    height: FULL,  // always FULL height in the DOM
    // The translateY controls how much is visible
    backgroundColor: '#1A1A1A',
    borderTopLeftRadius: 20,
    borderTopRightRadius: 20,
    overflow: 'hidden',
  },
  handleArea: {
    width: '100%',

    // Make the tap/drag target generously tall
    minHeight: 44,
  },
  handle: {
    width: 36,
    height: 4,
    borderRadius: 2,
    backgroundColor: '#555555',
  },

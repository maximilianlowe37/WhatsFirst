Both rows use the same pill style as the main filter bar:

  - Inactive: backgroundColor: 'transparent',

Wrap both rows in a View with:

**Filtering logic (computed from history tasks, no API call):**

  const filteredHistory = useMemo(() => {
    return historyTasks.filter(task => {
      // Status filter
      if (statusFilter !== 'all' && task.status !== statusFilter) 
        return false;
      
      // Date filter based on task.completedAt or task.cancelledAt
      const eventDate = new Date(task.completedAt || task.cancelledAt);
      const now = new Date();
      
      if (dateFilter === 'today') {
        return isSameDay(eventDate, now);
      }
      if (dateFilter === 'thisWeek') {
        const weekAgo = new Date(now);
        weekAgo.setDate(now.getDate() - 7);
        return eventDate >= weekAgo;
      }
      if (dateFilter === 'thisMonth') {
        return eventDate.getMonth() === now.getMonth() && 
               eventDate.getFullYear() === now.getFullYear();
      }
      return true; // 'all'
    });
  }, [historyTasks, statusFilter, dateFilter]);

  // Helper:
  function isSameDay(a, b) {
    return a.getFullYear() === b.getFullYear() &&
           a.getMonth() === b.getMonth() &&
           a.getDate() === b.getDate();
  }

**Grouping:**
After filtering, group by date of completedAt/cancelledAt

**Switching filters must be instant** — since filteredHistory

**Empty state:**
